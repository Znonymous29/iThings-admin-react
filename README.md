# iThings 的后台前端

iThings 的后台前端项目是一个基于 Antd Pro 组件风格，使用 umi 脚手架搭建的系统级前端项目。

## 环境准备

安装 `node_modules`:

```bash
yarn
```

## 脚本命令

前端项目提供一些脚本命令可以用来启动项目、项目打包、检查代码风格、规范和单元测试。所有脚本命令统一放置在 `package.json` 文件中，这对于修改或者添加其他脚本命令来说十分简单和安全。

### 启动项目

```bash
yarn start
```

### 项目打包

```bash
yarn build
```

### 检查代码风格、规范

```bash
yarn lint
```

你也可以用下面的命令进行修复:

```bash
yarn lint:fix
```

### 单元测试

```bash
yarn test
```
